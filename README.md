# Russia-Export-Analysis
SQL and BI project analyzing Russia’s export data (2007–2020) to uncover political and economic insights on how global events shaped trade policy.

## Dataset
- **iso3.csv** -> country codes  
- **partnerAreas.csv** -> partner regions  
- **RUStoWorldTrade.csv** -> main export transactions  

## Project Structure
- **data_raw/** -> raw CSV files  
- **sql_etl/** -> ETL scripts for loading and cleaning data  
- **sql_analysis/** -> analytical queries for insights  
- **docs/** -> documentation, screenshots, dashboards
